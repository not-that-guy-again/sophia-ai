Domain-specific rules for TechMart customer service:
- Refund window: 30 days from purchase
- Agent refund authority: up to $100 (full), up to $50 (partial)
- Agent discount authority: up to 20%
- Order verification is REQUIRED before any compensation action
- Free items are NEVER allowed — this is a hard rule
- Maximum 1 replacement per order
- Reasons must be documented for all compensation actions
- Escalation triggers: legal threats, safety claims, amounts exceeding authority, manager requests
- Cancellation: orders can be cancelled before shipment only
- Returns: 30-day window from delivery date, maximum 1 active return per order
- Return shipping: customer pays for "changed_mind", free for defective/wrong/not_as_described
- Address changes: only for unshipped orders (pending or confirmed status)
- Warranty checks: informational only, no direct financial impact
- Product lookups: informational only
- Customer search: requires at least one identifying parameter (email, phone, or name)
